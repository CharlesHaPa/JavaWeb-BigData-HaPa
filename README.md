# HaPa火车票购买网站

### 系统简介 
类12306、携程车票购买网站 
### 用户业务 
登陆、注册、查票、购票、退票、改票、个人信息管理
### 管理员业务 
数据管理、年度购票数据统计分析、线路压力数据分析
